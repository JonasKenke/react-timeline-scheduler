# Contributing to @jonaskenke/react-timeline-scheduler

Thank you for your interest in contributing to @jonaskenke/react-timeline-scheduler!

We use a Contributor License Agreement (CLA) to enable dual licensing of the code.

By contributing, you agree that your contributions can be licensed under both the AGPL v3 open source license and a separate commercial license.

Please review and sign the CLA before or during the pull request process.

## How to contribute

1. Fork the repository
2. Create a branch for your feature or bugfix
3. Commit your changes with clear messages
4. Push to your branch on GitHub
5. Open a Pull Request describing your changes

We appreciate well-tested, well-documented contributions.
